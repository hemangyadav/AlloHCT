# Using PFTs to individualize risk prediction prior to HCT
Research related to risk prediction for pulmonary complications after allogeneic hematopoietic stem cell transplantation

This code allows you to use your own data to visualize the relationship between pre-transplant FEV1, FVC and DLCO Z-scores and post-transplant survival, ICU utilization and specific outcomes (mechanical ventilation in our dataset).
